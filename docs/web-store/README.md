# Chrome Web Store assets

These files are publication assets for the Chrome Web Store dashboard. They are
not part of the packaged extension.

| File                                | Dashboard use           | Size       |
| ----------------------------------- | ----------------------- | ---------- |
| `z-s-promo.png`                     | Small promotional image | 440 x 280  |
| `zotero-speechify-screenshot.png`   | Screenshot              | 1280 x 800 |
| `zotero-speechify-screenshot-2.png` | Screenshot              | 1280 x 800 |

The extension icon is supplied by the packaged manifest from
`apps/chrome-extension/public/icons/icon-128.png`.

The optional marquee promotional image is not included.

Use [docs/privacy.md](../privacy.md) as the source for the Chrome Web Store
privacy-policy URL.
