# Privacy Policy

Last updated: July 9, 2026

Zotero Speechify is an unofficial Chrome extension for adding Speechify-based
voice controls to the Zotero Web reader. It is independent and is not
affiliated with, endorsed by, or supported by Speechify or Zotero.

The extension does not operate a server and does not include telemetry.

## Data processed

The extension processes only the data needed for its two user-initiated
workflows:

- Selected Zotero reader text, when the user asks the extension to read text
  aloud.
- Microphone audio and the resulting annotation transcript, when the user starts
  voice annotation.
- Zotero item and reader context needed to attach an annotation or create a
  child note.
- The user's Speechify API key and Zotero API key, stored in Chrome extension
  storage.

## External services

The extension sends data only to the services required for the selected
workflow:

- **Speechify** receives selected text for text-to-speech synthesis.
- **Speechify** receives microphone audio and transcript data during a voice
  agent annotation session.
- **Zotero** receives confirmed annotation or note content through
  `api.zotero.org`.

The extension does not send Speechify data to Zotero until the user confirms an
annotation, and it does not send Zotero API keys to Speechify.

## Storage

The user's Speechify API key, Zotero API key, selected voice settings, and
agent identifier are stored locally in Chrome extension storage. The extension
does not log API keys, microphone audio, transcripts, item metadata, or document
contents.

Confirmed annotations and notes are stored in the user's Zotero library. Zotero
sync may copy that data to the user's Zotero desktop installation and other
Zotero-connected devices.

## Retention and deletion

Data stored in the user's Zotero library can be edited or deleted by the user in
Zotero. Extension settings and API keys can be cleared from the extension's
Settings page or by removing the extension from Chrome.

Speechify and Zotero may retain data according to the user's account settings
and the applicable terms for those services. The extension does not control
third-party service retention.

## Changes

Privacy-relevant changes should be reflected in this policy before release.
