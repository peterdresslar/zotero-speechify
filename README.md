# Zotero Speechify

Zotero Speechify is an unofficial, independent Chrome extension experiment for
adding voice controls to Zotero's online reader. It is not affiliated with,
endorsed by, or supported by Speechify or Zotero.

The current scaffold focuses on the Chrome extension interface for the Zotero
Web reader. Say and Annotate API wiring will be added behind the existing
reader, popup, options, and command surfaces.
